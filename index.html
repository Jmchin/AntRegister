Ant(eater) Register

Ant Register is a console-based application built on Java. The program acts as a registration medium for students to view a master course registry, and add or drop courses from their profile.

To use Ant Register, please download the file "team-c-fall-2016.zip" and extract it to its own directory.

  • unzip team-c-fall-2016.zip to its own folder
  • verify that you have a "res/" directory and the .jar file together
  • open up a cmd prompt and navigate to the directory
  • java -jar team-c-fall-2016.jar
